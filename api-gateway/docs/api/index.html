# validatePassword(password, passwordRules)

Validates the password against a set of password rules specified to determine if it meets the criteria.

Parameters

Name Type Description
password string
  • The password that needs to be validated.
passwordRules object
  • An object representing the password rules.
passwordRules.minLength number
  • The minimum length of the password (absolute minimum is always 6 characters).
passwordRules.maxLength number
  • The maximum length of the password.
passwordRules.minDigits number
  • The minimum number of digits the password must contain (absolute minimum is always 1).
passwordRules.minSpecialCharacters number
  • The minimum number of special characters (absolute minimum is always 1).
passwordRules.minLowercaseLetters number
  • The minimum number of lowercase letters (absolute minimum is always 1).
passwordRules.minUppercaseLetters number
  • The minimum number of uppercase letters (absolute minimum is always 1).

Returns

object

A list of criteria that the password does not satisfy.

# sendGqlRequest(endpointUrl, gqlQueryString, gqlQueryVariables)

Sends a GQL query to the specified endpoint and returns the JSON data returned from the endpoint.

Parameters

Name Type Description
endpointUrl string
  • The GQL microservice endpoint to query.
gqlQueryString string
  • The GQL query/mutation to send to the microservice endpoint (i.e endpointUrl).
gqlQueryVariables object
  • A JS object containing string key/value pairs specifying different query variables to be sent to the endpoint.

Examples

Send a GQL request to the endpoint hosted on http://192.168.1.114:3002/api.

const response = sendGqlRequest('http://192.168.1.114:3002/api', gqlQueryString, gqlQueryVariables);
response
  .then((data) => {
    // Do something with the returned data
  })
  .catch((err) => {
    const { message } = err
    // Do something with the error and its message
  });

# rateLimiter([maxRequestWindowMs, maxRequestsAllowedPerWindow], onMaxRequestsExceeded)

Allows a maximum number of requests to the server from a client within a certain timeframe. Uses defaults of (maxRequestWindowMs = 1000, maxRequestsAllowedPerWindow=200).

Parameters

Name Type Description
maxRequestWindowMs number
  • The length of the rate limiting time window in milliseconds.
 Optional
maxRequestsAllowedPerWindow number
  • The maximum amount of requests that can be made within the time window.
 Optional
onMaxRequestsExceeded Function
  • The function that will run when the maximum number of requests have been exceeded by the client.

Examples

Only allow 10 requests per minute: rateLimiter(60000, 10, onMaxRequestsExceeded)

Returns
