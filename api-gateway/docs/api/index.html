# origin(origin, onOriginWhitelistCheck)

Checks the request's origin URL against a whitelist of domains to determine if its allowed to send CORS requests.

Parameters

Name Type Description
origin string
  • The requester UrL.
onOriginWhitelistCheck Function
  • The function that will called when the origin check has occured. It will return an error (if origin is not whitelisted) and a boolean indicating whether the origin is allowed to send CORS requests.

Returns

boolean

  • Whether the origin exists in the whitelisted or not.

# rateLimiter(maxRequestWindowMs, maxRequestsAllowedPerWindow, onMaxRequestsExceeded)

Allows a maximum number of requests to the server from a client within a certain timeframe.

Parameters

Name Type Description
maxRequestWindowMs number
  • The length of the rate limiting time window in milliseconds.
maxRequestsAllowedPerWindow number
  • The maximum amount of requests that can be made within the time window.
onMaxRequestsExceeded Function
  • The function that will run when the maximum number of requests have been exceeded by the client.

Examples

Only allow 10 requests per minute: rateLimiter(60000, 10, onMaxRequestsExceeded)

Returns

# getRequestOriginIpInfo(req)

Returns the IP address of the client that requested the resource (i.e the origin IP).

Parameters

Name Type Description
req express.Request
  • The express request object that is being recieved from the client.

Properties

Name Type Description
jsonObject.ip string
  • The IPv2 or IPv6 address.
jsonObject.timezone string
  • "Country/Zone" Timezone from IANA timezone database (https://www.iana.org/time-zones).
jsonObject.country string
  • 2 letter ISO-3166-1 country code from the IBAN country-codes database(https://www.iban.com/country-codes).

Returns

jsonObject

A JSON object representing the request's origin IP info.
